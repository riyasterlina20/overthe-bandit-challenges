# OverTheWire Bandit Challenges – Write-ups

This repository contains my structured write-ups for the **OverTheWire Bandit** wargame.
The purpose of this repository is to document my learning process while practicing
Linux basics, command-line usage, and fundamental cybersecurity concepts.

Each Bandit level is documented in a separate markdown file with:
- The objective of the level
- Commands used
- Step-by-step explanation
- outcomes
